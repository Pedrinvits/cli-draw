
# Cli-draw

Your Cli Task Manager

## Installation

```bash
npm install todo-cli
```
## Usage
```bash
npm start

npm add - to add new task

npm list - to list tasks

npm done - mark tasks as completed

npm remove - remove tasks

npm export - export the task list to a file
```
